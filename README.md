# 多人連線貪吃蛇遊戲

這是一個基於 Node.js 和 Socket.io 開發的即時多人連線貪吃蛇遊戲。

## 運行步驟

1. **安裝 Node.js**：確保你的電腦上安裝了 [Node.js](https://nodejs.org/)。
2. **解壓檔案**：將所有遊戲檔案放在同一個資料夾中。
3. **安裝依賴**：在該資料夾打開終端機（Terminal/CMD），執行：
   ```bash
   npm install
   ```
4. **啟動伺服器**：執行：
   ```bash
   npm start
   ```
5. **開始遊戲**：
   - 打開瀏覽器並訪問 `http://localhost:3000`。
   - 你可以打開多個分頁，或者讓同一區域網路下的朋友訪問你的 IP 地址，即可實現多人同場競技！

## 遊戲功能
- **即時同步**：所有玩家的位置和食物狀態在伺服器端統一計算。
- **多人競技**：支持多個玩家同時在線。
- **動態粗細**：蛇的身體會隨長度增加而變粗。
- **碰撞判定**：撞到其他玩家的身體會導致重生。
